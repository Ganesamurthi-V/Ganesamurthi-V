## 🚀 Projects

### 🏋️ [GymDesk — Gym Management SaaS Platform](https://github.com/Ganesamurthi-V/Gym-Management-System)
> **Next.js · TypeScript · PostgreSQL · Supabase**

A multi-module SaaS platform for gym owners with member management, attendance tracking, subscription billing, and business analytics dashboards. Features role-based access control (RBAC) and a scalable multi-tenant architecture.

---

### ⚡ [LightBase — Open Source Backend Platform](https://github.com/Ganesamurthi-V/LightBase)
> **TypeScript · PostgreSQL · REST APIs · Redis · BullMQ**

A self-hostable, open-source backend platform inspired by Supabase — providing authentication, database management, realtime, and API services out of the box. Built for extensibility and community contributions.

---

### 🧠 [Smart Career Guide — AI Resume & Job Matcher](https://github.com/Ganesamurthi-V/career-guidance-using-streamlit)
> **Python · NLP · Streamlit**

An AI-powered platform that analyzes resumes against job descriptions using NLP techniques to generate personalized job recommendations and skill gap reports.

---

### 🌡️ [Air Filter Prediction System](https://github.com/Ganesamurthi-V/Air-Filter-Status-Prediction-System)
> **Python · Flask · TSMixer · Machine Learning**

A production ML system to predict AC air filter health using the TSMixer time-series architecture. Deployed via Flask REST APIs for real-time monitoring and industrial integration. Built during internship at Iinvsys.
